# Portfólio Pessoal - Projeto de Faculdade

Este projeto é um website de portfólio pessoal, desenvolvido como parte dos requisitos acadêmicos da faculdade. Ele tem como objetivo apresentar minhas habilidades, projetos e experiências em desenvolvimento web.

## Funcionalidades

*   **Sobre Mim:** Apresentação pessoal.
*   **Formação Acadêmica:** Detalhes sobre a educação.
*   **Portfólio:** Uma galeria dos meus melhores projetos.
*   **Contato:** Informações para contato e redes sociais.

## Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript

## Como Visualizar

Este é um site estático. Para visualizá-lo localmente:

1.  Baixe ou clone o repositório.
2.  Abra o arquivo `index.html` em seu navegador web.

## Propósito do Projeto

Este portfólio foi criado para fins acadêmicos, como um projeto para a faculdade.
